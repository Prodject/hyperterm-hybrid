## Install

Open `~/.hyperterm.js` with your favorite editor and add `hyperterm-hybrid` to your plugins array

```js
plugins: ['hyperterm-hybrid']
```

Reload HyperTerm and Enjoy! ヾ(¯∇￣๑)
